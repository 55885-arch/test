========================================
วิธีเพิ่มบทเรียนใน data.json (อธิบายแบบง่าย)
========================================

ไฟล์ data.json คือไฟล์เก็บข้อมูลบทเรียนทั้งหมด
เวลาเพิ่มบทเรียนใหม่ ให้เพิ่มข้อมูลตามตัวอย่างนี้


ตัวอย่างการเพิ่ม 1 บทเรียน

{
  "title": "ชื่อเรื่อง",
  "subject": "ชื่อวิชา",
  "category": "-",
  "grade": "ม.1",
  "file": "FilesFolder/ชื่อไฟล์.pdf"
},


----------------------------------------
อธิบายแต่ละช่อง
----------------------------------------

"title"     = ชื่อบทเรียน
"subject"   = ชื่อวิชา
"category"  = หมวดย่อย (ถ้าไม่มีให้ใส่ "-")
"grade"     = ระดับชั้น เช่น ม.1
"file"      = ที่อยู่ไฟล์ PDF


----------------------------------------
รายชื่อวิชาที่ใช้ได้
----------------------------------------

คณิตศาสตร์
พื้นฐาน
ฟิสิกส์
เคมี
ชีววิทยา
ภาษาไทย
ภาษาอังกฤษ
สังคม


----------------------------------------
ระดับชั้นที่ใช้ได้
----------------------------------------

ม.1
ม.2
ม.3
ม.4
ม.5
ม.6


----------------------------------------
ตัวอย่างจริง
----------------------------------------

{
  "title": "แรงและการเคลื่อนที่",
  "subject": "ฟิสิกส์",
  "category": "-",
  "grade": "ม.4",
  "file": "FilesFolder/force.pdf"
},

{
  "title": "การอ่านจับใจความ",
  "subject": "ภาษาไทย",
  "category": "-",
  "grade": "ม.2",
  "file": "FilesFolder/thai.pdf"
}


----------------------------------------
สิ่งที่ต้องระวัง (สำคัญมาก)
----------------------------------------

1. ต้องสะกดชื่อวิชาให้ตรงตามรายการด้านบน
2. ต้องเขียน ม.1 แบบนี้เท่านั้น (ห้ามเขียน 1 หรือ m1)
3. ถ้าจะเพิ่มหลายบทเรียน
   ต้องใส่เครื่องหมาย , หลัง } ของอันก่อนหน้า
4. ห้ามใส่ , หลังตัวสุดท้ายในไฟล์
