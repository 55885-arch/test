========================================
วิธีเพิ่มบทเรียนในระบบ (อ่านง่ายสำหรับทุกคน)
========================================

ไฟล์ที่ต้องแก้ไขคือ: data.json
ไฟล์นี้เก็บข้อมูลบทเรียนทั้งหมดของเว็บ


----------------------------------------
ขั้นตอนเพิ่มบทเรียน
----------------------------------------

1. เปิดไฟล์ data.json
2. เลื่อนไปด้านล่างสุด
3. เพิ่มข้อมูลตามตัวอย่างนี้

{
  "title": "ชื่อเรื่อง",
  "subject": "ชื่อวิชา",
  "category": "-",
  "grade": "ม.1",
  "file": "FilesFolder/ชื่อไฟล์.pdf"
},

----------------------------------------
อธิบายแต่ละช่อง
----------------------------------------

"title"     = ชื่อบทเรียน
"subject"   = วิชา (ดูรายชื่อด้านล่าง)
"category"  = ถ้าไม่มีหมวด ให้ใส่ "-"
"grade"     = ชั้นเรียน เช่น ม.1
"file"      = ชื่อไฟล์ที่อยู่ในโฟลเดอร์ FilesFolder

----------------------------------------

{
  "title": "แรงและการเคลื่อนที่",
  "subject": "ฟิสิกส์",
  "category": "-",
  "grade": "ม.4",
  "file": "FilesFolder/force.pdf"
},


----------------------------------------
หลังจากเขียนเสร็จ ต้องทำอะไรต่อ?
----------------------------------------

1. กดบันทึกไฟล์ data.json
2. กลับไปที่หน้าเว็บ index.html
3. กดปุ่ม Refresh (รีโหลดหน้าเว็บ)
4. รอประมาณ 2-3 วินาที
5. บทเรียนใหม่จะขึ้นอัตโนมัติ


----------------------------------------
สิ่งที่ต้องระวัง
----------------------------------------

- ต้องสะกดชื่อวิชาให้ตรง
- ต้องเขียน ม.4 แบบนี้เท่านั้น
- ถ้าเพิ่มหลายรายการ ต้องใส่ , หลัง } ของรายการก่อนหน้า
- ห้ามใส่ , หลังรายการสุดท้ายในไฟล์
